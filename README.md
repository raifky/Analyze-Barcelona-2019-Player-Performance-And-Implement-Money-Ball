# Project Title
Analyze Barcelona 2019 Player Performance And Implement Money Ball
## Repository Outline

This project consist of 1 notebook, 1 main dataset 'Final_player.csv', 1 clean dataset 'clean_dataset.csv', description.md consist of description of this project, and there are 3 csv consist of data for ranking player of each position and 3 file csv consist of data candidate player of each position, and 1 file png consist of dataset screenshoot.

## Problem Background

As team owners from Barcelona, ​​last year we failed to win the La Liga or Spanish League championship against our rivals, namely Real Madrid. We failed to win the La Liga because there were several players who did not perform in several lines, for example in positions (defender, midfield and forward) therefore we failed to win the La Liga championship when the season was over. We as a team and the owner of Barcelona held a meeting to evaluate the Barcelona squad for next season so here our goal is to analyze which players will stay and which players will be released. This meeting measures several statistical factors for the defensive, attacking and central players. This meeting is very important because it will have an impact on our performance and the transfer of replacement players. If a player leaves next season, we need 30 days to release the player into the market window.

## Project Output

This project produces decision-making data for Barcelona to improve next season by using a method of correlating old players and players who will be bought.

## Data

The amount of data from the Final_player.csv file is 518 rows, 45 columns and there are missing values ​​in all the data for players who play goalkeepers, and this dataset is in CSV format.

## Method

The methods used to analyze this project include descriptive statistics such as calculating Mean, Median, Modus, Stdev, seeing skewness of data, seeing kurtosis of data, carrying out inferential statistics using the Spearman method and visualizing the data using the Python library.

## Stacks

The programming language i use in this project is python with liblary matplotlib, seaborn, spearman for claculating statistic inferential.
